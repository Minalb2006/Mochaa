JavaScript Conditional Statements

1. if Statement

the JavaScript if statement to execute a block of code when a condition is true.

let age = 20;
if (age >= 18) {
  console.log("You are an adult.");
}

2. if...else Statement

the else statement to specify a block of code to be executed if a condition is false

let score = 45;
if (score >= 50) {
  console.log("You passed.");
} else {
  console.log("You failed.");
}

3. else if Statement

The else if statement in JavaScript is used when you want to check multiple conditions one after another. 

let marks = 85;
if (marks >= 90) {
  console.log("Grade: A");
} else if (marks >= 75) {
  console.log("Grade: B");
} else {
  console.log("Grade: C");
}

4. switch Statement

he switch statement evaluates an expression and compares its result to multiple case values. When a match is found, the corresponding block of code runs. Itâ€™s a cleaner alternative to writing many if...else if statements. 

let day = "Monday";
switch (day) {
  case "Monday":
    console.log("Start of the week.");
    break; 
  case "Friday":
    console.log("End of the week.");
    break;
  default:
    console.log("Midweek day.");
}

5. Boolean Example

In JavaScript, a Boolean is a primitive data type that can only have one of two values: true or false The Boolean value of an expression is the basis for all JavaScript comparisons and conditions

let isLoggedIn = true;
if (isLoggedIn) {
  console.log("Welcome back!");
}

6. Logical Operators

Logical operators are used to combine boolean expressions. Logical operators can be used to modify the results of comparisons.

// AND (&&)
let age = 25;
let hasID = true;
if (age >= 18 && hasID) {
  console.log("Access granted.");
}

// OR (||)
let isWeekend = true;
let isHoliday = false;
if (isWeekend || isHoliday) {
  console.log("You can relax today.");
}

// NOT (!)
let isRaining = false;
if (!isRaining) {
  console.log("Go for a walk.");
}
