JavaScript Strings

1. String introduction

A String in JavaScript is a sequence of characters used to represent text such as words or sentences. Strings are enclosed in single quotes (' '), double quotes (" "), or backticks ( ).

Example :  let name = "Minal";
console.log("My name is " + name);


output: My name is Minal

2. String templates

Template Literals are a special way to create strings in JavaScript. They allow you to insert variables, expressions, or even multi-line text directly inside a string without using the + operator. Template literals make your code simpler, cleaner, and more readable. Use backticks (`) instead of single or double quotes. string text ${expression}

Example : Inserting Variables
let name = "Minal";
let age = 20;
console.log(My name is ${name} and I am ${age} years old.);

Output:
My name is Minal and I am 20 years old.

3. String Methods

1. .length: Returns the number of characters in a string.

 example: "hello".length;
    output: // 5

2. .toUpperCase(): Converts the string to uppercase.

example:"hello".toUpperCase(); 
    output: // "HELLO"

3. toLowerCase(): Converts the string to lowercase.

example: "HELLO".toLowerCase();
     output: // "hello"

4. charAt(index): Returns the character at the given index.

example: "hello".charAt(1); 
    output: // "e"

5. charCodeAt(index): Returns the Unicode value of a character.

example: "A".charCodeAt(0); 
    output: // 65

6. at(index): Returns the character at a position (also supports negative indexing).

example: "hello".at(-1); 
    output: // "o"

7. slice(start, end): Extracts part of the string (end not included).

example: "JavaScript".slice(0, 4); 
    output: // "Java"

8. substring(start, end): Similar to slice() but doesn’t support negative indexes.

example: "JavaScript".substring(4, 10); 
    output: // "Script"

9. substr(start, length): Extracts a substring of given length (older method).

example: "JavaScript".substr(0, 4);
    output:  // "Java"

10. concat(): Joins two or more strings.

example: "Hello".concat(" ", "World"); 
    output: // "Hello World"

11. trim(): Removes spaces from both ends of a string.

example: " Hello  ".trim(); 
    output: // "Hello"

12. trimStart() / trimEnd(): Removes spaces from start or end only.

example: "Hi".trimStart(); 
    output: // "Hi"

13. replace(old, new): Replaces the first match with new text.

example: "apple".replace("a", "A"); 
    output: // "Apple"

14. replaceAll(old, new): Replaces all occurrences of text.

example: "banana".replaceAll("a", "A"); 
    output: // "bAnAnA"

15. split(separator): Splits a string into an array.

example: "red,green,blue".split(","); 
    output: // ["red", "green", "blue"]

16. includes(substring): Checks if the string contains a substring → returns true/false.

example: "JavaScript".includes("Script"); 
    output: // true

17. startsWith(substring): Checks if a string starts with given substring.

example: "Hello".startsWith("He"); 
    output: // true

18. endsWith(substring): Checks if a string ends with given substring.

example: "Hello".endsWith("lo"); 
    output: // true

19. indexOf(substring): Returns the first position of substring.

example: "banana".indexOf("a"); 
    output: // 1

20. lastIndexOf(substring): Returns the last position of substring.

example: "banana".lastIndexOf("a"); 
    output: // 5

21. repeat(count): Repeats the string given number of times.

example: "ha".repeat(3); 
    output: // "hahaha"

22. valueOf(): Returns the primitive value of a string object.

example: "Hello".endsWith("lo"); 
    output: // true

23. toString(): Converts any value to a string.

example: (123).toString(); 
    output: // "123"

4. String search Methods

These methods are used to search for characters, words, or patterns inside strings.

1. indexOf(searchValue, start): Returns the first position (index) of the given text.

 example: "hello world".indexOf("o"); 
    output: // 4

2. lastIndexOf(searchValue, start): Returns the last position of the given text.Searches from the end. 

example: "hello world".lastIndexOf("o"); 
    output: // 7

3. search(pattern): Searches for a substring or regular expression.Returns the index of the first match. 

example: "JavaScript".search("Script"); 
    output: // 4

4. match(pattern): Finds matches of a regular expression.Returns an array of results or null if no match. 

example: "abc123".match(/\d+/); 
    output: // ["123"]

5. matchAll(pattern): Returns all matches (as an iterator).Used with global regex /g. 

example: "test123test456".matchAll(/\d+/g);
output: // "123", "456"

6.includes(searchValue, start): Returns true if the string contains the given text.Case-sensitive.

example: "Hello".includes("He"); 
    output: // true

7. startsWith(searchValue, start): Checks if the string begins with the given text.Returns true or false.

example: "JavaScript".startsWith("Java"); 
    output: // true

8. endsWith(searchValue, length): Checks if the string ends with the given text.Returns true or false. 

example: "JavaScript".endsWith("Script"); 
    output: // true
